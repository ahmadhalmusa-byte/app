ScanFlow - Privacy Policy

Last Updated: April 2026

1. Introduction

ScanFlow is a barcode and QR code scanner Android application. This Privacy Policy describes how we handle user data.

2. Data Collection

ScanFlow operates entirely offline and does NOT collect any personal information.

The app does NOT require internet connection to function.

3. Local Storage Only

All scan data is stored locally on the user's device only:

  • Scanned barcode/QR code content
  • Scan timestamps
  • User-added names and notes
  • Category assignments
  • Favorite status

This data remains on the device and is NEVER transmitted to external servers.

4. On-Device Processing

The app uses Google ML Kit for barcode scanning. All processing happens entirely on-device – no data is sent to Google or any third-party servers.

5. Permissions Required

  • Camera: Used exclusively for scanning barcodes and QR codes
  • Vibration: Provides feedback when a scan is successful

6. No Children Data Collection

ScanFlow does NOT knowingly collect information from children under 13.

7. No Advertising or Tracking

The app does NOT contain any advertising or tracking functionality.

8. Changes to This Policy

This privacy policy may be updated. Any changes will be reflected in the documentation included with the app.

9. Contact

For privacy-related questions, please use the GitHub repository issues page.
